Data Sources
  • IEA — Oil Market Report (monthly supply/demand estimates)
  • EIA — Short-Term Energy Outlook (U.S. + global production data)
  • OPEC — Monthly Oil Market Report (OPEC member output & quotas)

Note: All figures are 2024 annual averages. Minor discrepancies between sources reconciled manually.

Methodology
  • Country-level data from IEA, EIA, OPEC MOMR
  • XLOOKUP links raw data to balance sheet dynamically
  • SUMIF / SUMIFS aggregate by group (OPEC / Non-OPEC)
  • Pivot Tables produce group-level summaries
  • Scenario engine uses named ranges + IF logic
  • Price bias = f(net balance, cut size, geo premium)
  • Dashboard auto-updates via cell references

Net Balance = Production + Imports − Consumption − Exports. A positive value indicates a surplus (bearish signal); negative indicates a deficit (bullish signal).
